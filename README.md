🎨 告知画像ジェネレーター (SNS Image Maker)

部活の試合、文化祭、イベントなどの告知画像を、誰でもカンタンに作れるツールです。
日付・時間・タイトルを入力するだけで、InstagramのストーリーやX（旧Twitter）に投稿できる画像が完成します。

✨ できること

カンタン作成: 日付、時間、タイトルを入力するだけ。

3つのデザイン: 「ポップ」「クール」「エレガント」から選べます。

すぐに保存: ボタンひとつで画像をダウンロードできます。

インストール不要: ファイルを開くだけで動きます。

🚀 使い方

起動する

フォルダの中にある index.html をダブルクリックしてください。ブラウザ（Chromeなど）が開きます。

入力する

画面の左側にある入力欄に、日付やタイトルを入れます。

デザインを選びます。

確認する

右側のプレビュー画面で、仕上がりを確認します。文字を変えるとリアルタイムで更新されます。

保存する

「画像をダウンロード」ボタンを押すと、画像ファイル（PNG）として保存されます。

あとはSNSに投稿するだけ！

🛠 改造したい人へ（中級者向け）

このアプリは HTML / CSS / JavaScript だけで作られています。
index.html を VSコードなどのエディタで開くと、中身を改造できます。

色を変えたい: drawPopDesign などの関数の中にある ctx.fillStyle の色コードを変えてみよう。

サイズを変えたい: <canvas> タグの width と height を変更すると、画像のサイズが変わります（現在は 1080x1920 のスマホ縦長サイズです）。

📝 その他

このアプリはブラウザ上で動くため、入力した内容がどこかのサーバーに送信されることはありません。安心して使ってください。
